# CMA Study Tracker — Android App

A native Android app for CMA exam preparation.

## Features
- Dashboard with exam countdown
- Topic List with study dates & times
- Daily Scheduler — shows all chapters active on each day
- Full notifications at scheduled study times
- Do Not Disturb reminders during focus hours
- Dark mode

---

## How to Build the APK (No coding needed)

### Step 1 — Upload to GitHub

1. Go to [github.com](https://github.com) and sign in (or create free account)
2. Click **"+"** → **New repository**
3. Name it `cma-tracker`, set to **Public**, click **Create repository**
4. Click **"uploading an existing file"**
5. Drag and drop ALL files from this zip (keeping the folder structure)
6. Click **"Commit changes"**

### Step 2 — Wait for automatic build (~5 minutes)

1. Click the **"Actions"** tab in your repository
2. You will see **"Build CMA Tracker APK"** running
3. Wait for the green ✅ checkmark

### Step 3 — Download your APK

1. Click on the completed workflow run
2. Scroll down to **"Artifacts"**
3. Click **"CMA-Tracker-APK"** to download
4. Unzip → you get `app-release.apk`

### Step 4 — Install on your phone

1. Send the APK to your phone (WhatsApp, email, Google Drive)
2. On Android: **Settings → Security → Install unknown apps** → Allow
3. Tap the APK → Install ✅

---

## After installing

### Enable notifications
- Allow notifications when the app asks
- Go to **Settings tab** → **Reschedule All Notifications**

### Enable Do Not Disturb
- Go to **Settings tab** → **Do Not Disturb During Study** → Toggle ON
- Set your study start and end time
- When your study time arrives, a notification will appear
- Tap it → go to **Android DND Settings** → set to **Priority Only → Allow Calls**
